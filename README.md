# Resource-Aware Neural Architecture Adaptation

This project explores adaptive neural networks that adjust their computation based on available system resources such as memory and compute.

## Goals
- Build a transformer with early exit points
- Detect system resource constraints
- Adapt computation depth dynamically
- Evaluate performance across different resource conditions

## Tech Stack
- PyTorch
- Vision Transformers
- Reinforcement Learning (yet to decide if we're gonna do it or not)
